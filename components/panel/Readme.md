## 基本用法
```
<woo-panel style="height: 100px">
  默认容器包含上下两个边框
</woo-panel>
```

## 边框
```
<woo-panel border="all" style="height: 100px">
  全边框
</woo-panel>
<woo-panel border="left, right" style="height: 100px">
  左右边框
</woo-panel>
<woo-panel border="top, left, right" style="height: 100px">
  上左右边框
</woo-panel>
<woo-panel border="none" style="height: 100px">
  无边框
</woo-panel>
```

## 边框色
```
<woo-panel border="all" border-color="#f00">
  全边框
</woo-panel>
```

## 背景
```
<woo-panel border="all" background-color="#f0f0f0">
  全边框
</woo-panel>
```
